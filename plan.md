分享一些我常用的软件，附有简单的介绍和使用技巧。

这里分享的是适合广大普通用户的软件，所以一些用途很少的，以及不适合大众的（如编程类）的软件不做推荐。

# 分类

- 下载软件
- 压缩软件
- 截屏录屏
- 图片处理
- 音乐视频
- 系统增强

# 每个软件的字段

名称
分类
链接
详情

# 每个软件的信息如何存放

图片文件放在 images 文件夹

数据存储在 redis 数据库里

# API

## 获取所有分类

```/category/all```

## 获取指定分类的所有软件

```/category/{name}```

## 获取首屏展示软件

```/default```

目前是打算从数据库里调出前 n 个软件的信息。

## 获取某个软件的详情

```/{name}```

## 搜索功能

```/search/{word}```
